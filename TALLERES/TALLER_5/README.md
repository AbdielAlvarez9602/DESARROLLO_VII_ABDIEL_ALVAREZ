Sistema de Gestión de Estudiantes

Este proyecto es un Sistema de Gestión de Estudiantes que permite gestionar información sobre estudiantes, sus calificaciones y carreras, y cuenta con funcionalidades útiles para administrar y analizar los datos.

Funcionalidades

1. Gestión de Estudiantes
Agregar Estudiantes: Puedes añadir nuevos estudiantes con su información básica, como nombre, edad, carrera y calificaciones en diferentes materias.

2. Calificaciones y Promedios
Agregar Materias y Calificaciones: Para cada estudiante, puedes agregar materias junto con sus calificaciones.
Calcular Promedio: El sistema calcula automáticamente el promedio de calificaciones de cada estudiante.

3. Reportes
Reporte de Rendimiento: Puedes generar un informe que muestra el promedio de calificaciones, la calificación más alta y la más baja para cada materia.

4. Búsqueda y Filtrado
Buscar Estudiantes: Puedes buscar estudiantes por nombre o carrera, incluyendo coincidencias.
Listar Estudiantes por Carrera: Filtra estudiantes según su carrera.

5. Estadísticas
Estadísticas por Carrera: Muestra cuántos estudiantes hay en cada carrera, su promedio general y quién es el mejor estudiante de cada carrera.

6. Graduación
Graduar Estudiantes: Puedes "graduar" a un estudiante, lo que significa que se elimina del sistema y se guarda en un registro de graduados.

7. Ranking
Generar Ranking: El sistema puede ordenar a los estudiantes según su promedio y generar un ranking.
